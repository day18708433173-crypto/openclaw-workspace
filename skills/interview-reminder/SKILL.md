# 面试提醒助手

每天 09:00（Asia/Shanghai）生成当天面试清单和待安排面试清单。只通知面试事项，不加入其他工作安排。

## 数据源

使用 [TOOLS.md](../../TOOLS.md) 中的招聘智能表：

- `file_id=DUFRmZHBzTlRBUWJucWJSeE9Q`
- `sheet_id=BB08J2`
- 一面：`一面时间`、`一面面试官`、`一面结果`
- 二面：`二面时间`、`二面面试官`、`二面结果`
- 身份信息：`候选人姓名`、`应聘岗位`

## 执行流程

1. 以 `Asia/Shanghai` 计算今天的 `00:00:00` 至 `23:59:59.999`。
2. 直接调用 `mcporter` 拉取智能表全部记录；`has_more=true` 时继续翻页，不修改表格。

```bash
mcporter call tencent-saas-docs "smartsheet.list_records" file_id:DUFRmZHBzTlRBUWJucWJSeE9Q sheet_id:BB08J2 limit:100
```

3. 将完整响应保存为无 BOM 的 UTF-8 JSON 文件，调用确定性脚本：

```powershell
node skills/interview-reminder/scripts/build-reminder.js --json-file <records.json> --timezone Asia/Shanghai
```

测试或补跑历史日期时增加 `--date YYYY-MM-DD`。

4. 将脚本标准输出原样作为最终消息，不补充天气、会议、非面试待办或统计分析。

## 筛选规则

- 一面时间在当天且一面结果不是 `通过`、`未通过`、`取消`：提醒一面。
- 二面时间在当天且二面结果不是 `通过`、`未通过`、`取消`：提醒二面。
- 记录已进入招聘表、一面时间为空且一面结果为空：视为初筛通过，提醒安排一面。
- 一面结果为 `通过`、二面时间为空且二面结果为空：提醒安排二面。
- 一面结果为 `未通过` 或 `取消`：不提醒安排二面。
- 同一候选人同一天有两轮时分别列出。
- 按时间升序；缺失面试官时显示 `⚠️ 未填写`。
- 无已安排面试但有待安排事项时，仍发送待安排清单。
- 两类事项都为空时仅输出 `今日无已安排面试，也无待安排事项。`

## 输出格式

```text
📅 面试事项提醒｜7月20日

【今日面试｜2场】

• 10:00｜一面｜张三（产品经理）
  面试官：王敏
• 15:30｜二面｜李四（前端开发）
  面试官：赵强

【待安排面试｜2人】
• 王五（后端开发）｜初筛通过，待安排一面
• 赵六（测试工程师）｜一面通过，待安排二面
```

## 铁律

- 只读，不写招聘表。
- 只通知面试事项。
- 不臆测缺失的时间、候选人、岗位或面试官。
- 使用脚本筛选和排版，不手工计算日期边界。
